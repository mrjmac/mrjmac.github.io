Discord Wordle Bot

  • Built completely in JavaScript using discord.js
  • Automatically kept track of the average attempts it took all users on the server to solve wordle

VALORANT Automatic Video Editor

  • A Python script that uses AI to turn any VALORANT video into a shorter one containing only moments where the person recording got a kill
  • Written 100% in Python with the use of OpenCV, moviePy, and YOLOv5
  • Uses a neural network that was trained for 1000 epochs on an 8000 image dataset labeled by me over a 3 day time period

YouTube Shorts AskReddit Video Creator

  • Automatically creates a video which reads out the comments in a popular AskReddit thread over background gameplay, just like popular youtube shorts and tiktoks do.
  • Written 100% in Python
  • Uses the PRAW Reddit library to find posts
  • Takes a screenshot of the post using a DOM based web scraper
  • Uses Google Text to Speech (gTTS) to generate audio
  • Uses pytube to automatically download a youtube video
  • Splices everything together with moviePy
  • Contains an additional file to automatically upload generated videos to YouTube
