Square Pasture

GitHub link

This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link to it so you can read it.

This problem is very simple and is pure math. We first need to find the x and y coordinates of the most extreme points given. This is the minimum values for the left and bottom points and the maximum values for the top and right points. Now we have all we need to make the smallest rectangular pasture, however the problem asks for a square pasture. To make a square pasture all we have to do is find the largest side of the rectangle and return that squared. Here's the code for this problem:

int low = min(x1, x3);
int low2 = min(y1, y3);
int high = max(x2, x4);
int high2 = max(y2, y4);

cout << max(high - low, high2 - low2) * max(high - low, high2 - low2) << "\n";
