This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem isn't very bad, as you are given the algorithm needed so solve it in the problem description. All I had to do was implement it. At first I forgot to account for test cases where Farmer John was ahead of Bessie, however once I fixed this the problem was extremely straight forwards. Here's the code for this problem:

while (!found)
{
    if ((dir == 1 && x <= y && y <= x + distance) || (dir == -1 && x >= y && y >= x - distance))
    {
        answer += abs(y - x);
        cout << answer << "\n";
        found = true;
    }
    else
    {
        answer += distance * 2;
        distance *= 2;
        dir *= -1;
    }
}
