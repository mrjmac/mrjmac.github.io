Coin Piles

GitHub link

This is the eleventh problem in the CSES problem set.

The problem goes like this: You have two coin piles containing a and b coins. On each move, you can either remove one coin from the left pile and two coins from the right pile, or two coins from the left pile and one coin from the right pile. Figure out if you can empty both piles.

This problem is a lot easier than it seems. The first thing I noticed is that it will always be impossible the empty the stack if one if more than twice as big as the other. This is because even if you opt to remove 2 coins from the biggest stack on every turn, the other stack will be emptied too fast.

To figure out what other rule I could use to check if piles could be emptied, I filled the piles from 0 instead of emptying them. Here were the possibilities I came up with:

1 2

3 3

2 4

I noticed after writing these possibilities down that no matter how I add coins to the piles, 3 coins are being added overall. Therefore, if a given pile meets the first rule in the above paragraph and is divisible by 3, the pile can be emptied. Here's the code for this problem:

if (a > b)
{
    if (b * 2 < a)
    {
        pw.println("NO");
    }
    else if ((a + b) % 3 == 0)
    {
        pw.println("YES");
    }
    else {
        pw.println("NO");
    }
}
else
{
    if (a * 2 < b)
    {
        pw.println("NO");
    }
    else if ((a + b) % 3 == 0)
    {
        pw.println("YES");
    }
    else {
        pw.println("NO");
    }
}
