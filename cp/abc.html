This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link to it so you can read it.

This problem is more math. We know that the two smallest values with be a and b. We know that the largest value will be the sum of a, b, and c. Therefore, we can put all the given values into an array and sort it. Once this is done, we can print out our first and second values in the array as those are a and b, and then we can print out the final value minus the first two as that will equal c. This is because (a + b + c) - a - b reduces to c. Here's the code for this problem:

for (int i = 0; i < 7; i++)
{
    nums[i] = Integer.parseInt(st.nextToken());
}
Arrays.sort(nums);
pw.println(nums[0] + " " + nums[1] + " " + (nums[6] - nums[1] - nums[0]));
pw.close();
