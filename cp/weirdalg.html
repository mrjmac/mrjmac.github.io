This is the first problem in the CSES competitive programming problem set. I've decided to work my through this set in order to get better at competitive programming so that I can succeed in USACO and UIL Computer Science.

The problem goes like this: You are given a positive integer N from 1 to 10^6. If N is even, you divide it by 2. If N is odd, you multiply it by 3 and add 1. Print out all values N is ever equal to before it hits 1.

This problem is pretty simple and you'll probably find the solution right away. The easiest way to solve this is by simulating the value of N until it hits 1. However, there it one element that it is easy to forget to consider here. Most people will take N in as an integer, as 10^6 is far under the maximum integer value. However, some values of N will exceed the maximum integer limit as they get multiplied by 3 repeatedly. This is easily fixed by storing N as a long. Here's the code for this problem:

long a = input.nextLong();
        String output = a + " ";
        while (a != 1)
        {
            if (a % 2 == 0)
            {
                a /= 2;
            }
            else 
            {
                a = (a * 3) + 1;
            }
            output += a + " ";
        }
        System.out.println(output);
