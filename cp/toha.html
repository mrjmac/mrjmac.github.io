GitHub link

This is the thirteenth problem in the CSES problem set.

This problem is classic and my explanation and writeup can never do the genius of the algorithm justice. Please try to solve it here and read this writeup when you're done.

Here's the code for the recursive method in this problem:

public static void hanoi(int n, int f, int t, int a, PrintWriter pw)
{
    if (n > 0)
    {
        hanoi(n - 1, f, a, t, pw);
        pw.println(f + " " + t);
        hanoi(n - 1, a, t, f, pw);
    }
}
