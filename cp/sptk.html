Speeding Ticket

This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem is deceptively simple. My first thought was to use math for this problem like I did in Mixing Milk, but that's actually a pretty hard approach to this problem. Instead, it's much easier to create an array that contains the speed limit for each of the 100 miles on the road. Then we make a secondary array that contains the speed that Bessie went for each of the 100 miles on the road. Then all we have to do is compare the two and find the maximum offset. This problem is an excellent example of how simulation can be much easier than math. Here's the code for this problem:

int last = 0;
vector<int> speedlimits(100);

for (int i = 0; i < n; i++)
{
    int length, speed;
    cin >> length >> speed;
    for (int k = last; k < last + length; k++)
    {
        speedlimits[k] = speed;
    }
    last += length;
}

last = 0;
vector<int> bessieSpeeds(100);

for (int i = 0; i < n; i++)
{
    int length, speed;
    cin >> length >> speed;
    for (int k = last; k < last + length; k++)
    {
        bessieSpeeds[k] = speed;
    }
    last += length;
}

for (int i = 0; i < 100; i++)
{
    speeding = max(speeding, bessieSpeeds[i] - speedlimits[i]);
}

cout << speeding;
