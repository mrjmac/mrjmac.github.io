This was a problem from the 2023 Westwood UIL invitational.

The problem goes like this: Given two numbers, output the sum.

The problem was as easy as it sounds. Here's the code:

public class Diem {
    public static void main(String args[]) throws IOException
    {
        Scanner test = new Scanner(new File("diem.dat"));

        int n = test.nextInt();

        for (int i = 0; i < n; i++)
        {
            int a = test.nextInt(), b =  test.nextInt();

            System.out.println(a + b);
        }


    }
}
