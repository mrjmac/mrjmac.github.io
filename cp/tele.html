This problem is incredibly easy and can be solved by computing every possibility as there is only ever 3 of them. Here's the code for this problem:

int ans = Math.abs(a - b);
ans = Math.min(ans, Math.abs(a - x) + Math.abs(b - y));
ans = Math.min(ans, Math.abs(a - y) + Math.abs(b - x));
pw.println(ans);
