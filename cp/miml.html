Mixing Milk

GitHub link

This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem is extremely trivial. As N is limited to 100, we can easily simulate every pour within the time limit. The index of the current bucket will be equal to i % 3. At first I added 1 to this index in order to access the bucket after it, however I realized this was wrong before I even tested my code. In the event of i == 2, we would try to access an index out of bounds with this setup. Instead, the index of the second bucket can be found by doing (i + 1) % 3. After we have our indexes, we just have to pour the most milk possible into the next bucket. This is equal to lower of the limit of the 2nd bucket minus the current amount of milk in the second bucket and the amount of milk in the first bucket. Here's the code for this problem

vector<int> limits(3);
vector<int> buckets(3);

cin >> limits[0] >> buckets[0] >> limits[1] >> buckets[1] >> limits[2] >> buckets[2]; 

for (int i = 0; i < 100; i++)
{
    int pos = i % 3;
    int pos2 = (i + 1) % 3; // prevent out of bounds index
    int milk = min(buckets[pos], limits[pos2] - buckets[pos2]);
    buckets[pos] -= milk;
    buckets[pos2] += milk;
}

for (int i = 0; i < (int)limits.size(); i++)
{
   cout << buckets[i] << "\n"; 
}
