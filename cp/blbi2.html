This problem is very simple but really annoying. My first thought was to cover the area of the lawnmower billboard minus the area of the intersection between the lawnmower billboard and the feed billboard. However, this doesn't account for the case where the lawnmower billboard has one side only partially covered by the feed billboard. Instead, I had to manually check and output the math for all 6 base cases. These cases are: the lawnmower billboard is completely covered, has it's left side covered, it's right side covered, it's top side covered, it's bottom side covered, and it's diagonal covered. Here's the code for this problem

cin >> x1 >> y1 >> x2 >> y2 >> x3 >> y3 >> x4 >> y4;

if (x4 >= x2 && x3 <= x1 && y4 >= y2 && y3 <= y1)
{
    cout << 0;
} 
else if (x3 < x2 && x4 >= x2 && y4 >= y2 && y3 <= x1) 
{ 
    cout << (x3 - x1) * (y2 - y1);
} 
else if (x3 <= x1 && y3 <= y1 && y4 > y1 && x4 >= x2) 
{
    cout << (x2 - x1) * (y2 - y4);
} 
else if (y3 < y2 && x3 <= x1 && y4 >= y2 && x4 >= x2) 
{
    cout << (x2 - x1) * (y3 - y1);
} 
else if (x4 > x1 && x3 <= x1 && y4 >= y2 && y3 <= y1) 
{
    cout << (x2 - x4) * (y2 - y1);
} 
else 
{
    cout << (x2 - x1) * (y2 - y1);
}
