Grappling Hook

GitHub link

This was a problem from the 2023 Westwood UIL invitational.

The problem goes like this: You are falling off a cliff and have to figure out what angle to aim your grapping hook at. However, you need to factor in the time it will take you to calculate the angle you need to aim it at to figure out the actual angle needed. The time it will take you to aim the grappling hook is the height you have fallen divided by 100 and you are moving at a terminal 50 m/s. What angle should you aim your hook at?

This problem was pretty easy. The math is pretty self explanatory and just requires knowledge of basic geometry. There were two edge cases, which were when the distance was either 0 or the height was 0. Here's the code for this problem:

public class Hook {
    public static void main(String args[]) throws IOException
    {
        Scanner test = new Scanner(new File("hook.dat"));

        int n = test.nextInt();
        test.nextLine();

        for (int i = 0; i < n; i++)
        {
            double d = test.nextDouble();
            double h = test.nextDouble();
            double t = h / 100;
            if (d == 0)
            {
                System.out.println("90.000");
            }
            else if (h == 0)
            {
                System.out.println("0.000");
            }
            else 
            {
                h = h + (50 * t);
                System.out.println(String.format("%.3f", Math.toDegrees(Math.atan(h / d))));
            }
        }
    }
}
