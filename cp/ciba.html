This problem is another one that you can simulate very easily. The only slight issue is figuring out how to loop around if you have to go from door n to door 1. This is pretty easy though, as this can be solved for by finding (starting room + current room) % number of rooms. Here's the code for this problem:

int mindis = INT32_MAX;
for (int i = 0; i < n; i++)
{
    int currdis = 0;
    int cows = totalcows;
    for (int j = 0; j < n; j++)
    {
        cows -= rooms[(i + j) % n];
        currdis += cows;
    }
    mindis = min(mindis, currdis);
}
