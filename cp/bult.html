This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem is pretty similar to the problem speeding ticket. It looks pretty complicated at first, however it's pretty easy once you realize that you can just add the time that any number of buckets are "checked out" to an array. After you've done this, you just have to loop through the array and keep track of how many buckets are checked out at one time. Here's the code for this problem:

for (int i = 0; i < n; i++)
{
    int start, end, buckets;
    cin >> start >> end >> buckets;

    times[start] += buckets;

    times[end] -= buckets;
}

int greatest = 0, cur = 0;
for (int i = 0; i <= 1000; i++)
{
    cur += times[i];
    greatest = max(greatest, cur);
}

cout << greatest << "\n";
