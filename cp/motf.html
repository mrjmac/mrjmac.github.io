This problem is a lot easier than it looks. At first I thought that the way to go about this would be do some math. However, it's actually pretty easy to simulate this. The max value that our answer can be is 1000 as Farmer John takes 10 steps maximum in a maximum of 100 actions. All that I had to do to solve this problem was to simulate the movement of Farmer John and mark the time that he visited each grass. Then if any piece of grass that he visited already had a value, the answer would be equal to the current time minus the previously marked time. Here's the code for this problem:

int main()
{
    cin.tie(0)->sync_with_stdio(0);
    freopen("mowing.in", "r", stdin);
    freopen("mowing.out", "w", stdout);

    int currX = 1000, currY = 1000, time = 0, ans = 1001;

    int n;
    cin >> n;

    for (int i = 0; i < n; i++)
    {
        string dir;
        int numSteps, dx = 0, dy = 0;
        cin >> dir >> numSteps;

        if (dir == "N")
        {
            dy = 1;
        }
        else if (dir == "S")
        {
            dy = -1;
        }
        else if (dir == "W")
        {
            dx = -1;
        }
        else
        {
            dx = 1;
        }

        for (int i = 0; i < numSteps; i++)
        {
            currX += dx;
            currY += dy;
            time += 1;
            if (field[currX][currY] > 0 && time - field[currX][currY] < ans)
            {
                ans = time - field[currX][currY];
            }
            field[currX][currY] = time;
        }
    }

    if (ans == 1001)
    {
        ans = -1;
    }
    cout << ans << "\n";
}
