This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link to it so you can read it.

This problem is very simple and is just pure math. In order to find the total length painted, we just need to add both of the original lengths before subtracting the length in the middle where the two lengths overlap. If we define the first length as A, B and the second length as C, D, then the overlap can only exist if B - C is positive. However, this problem does not guarantee that line AB comes before CD. In order to fix this, we have to take the minimum value of B and D as that value is the end of the first line segment. Additionally, we have to find the maximum value of A and C as that is the start of the second line segment. Here's the code for this problem:

int falseTotal = (b - a) + (d - c);
    int intersection = max(min(b, d) - max(a, c), 0);
    cout << falseTotal - intersection << "\n";
