This was a problem from the 2023 Westwood UIL invitational.

The problem goes like this: for each word in a list, print it out in alphabetical order.

This problem was very easy and was solved exactly how one would assume. For each word all letters were added to an array and sorted. Then they were printed out. Here's the code for this problem:

public class Vigenere {
    public static void main(String args[]) throws IOException
    {
        Scanner test = new Scanner(new File("vigenere.dat"));

        int n = test.nextInt();
        test.nextLine();

        for(int i = 0; i < n; i++)
        {
            String curr = test.nextLine();
            ArrayList<Character> sort = new ArrayList<>();

            for (int j = 0; j < curr.length(); j++)
            {
                sort.add(curr.charAt(j));
            }
            Collections.sort(sort);
            for (char c : sort)
            {
                System.out.print(c);
            }
            System.out.println();
        }


    }
}
