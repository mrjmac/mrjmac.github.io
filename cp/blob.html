This was a pretty simple problem that I created for myself after seeing something similar during my cs class. Given a 2D String array filled with @ symbols and - symbols, count the number of @ signs. 

String[][] blobs = {{"@", "-", "@", "-", "-", "@", "-", "@", "@", "@"},
                            {"@", "@", "@", "-", "@", "@", "-", "@", "-", "@"},
                            {"-", "-", "-", "-", "-", "-", "-", "@", "@", "@"},
                            {"-", "@", "@", "@", "@", "@", "-", "@", "-", "@"},
                            {"-", "@", "-", "@", "-", "@", "-", "@", "-", "@"},
                            {"@", "@", "@", "@", "@", "@", "-", "@", "@", "@"},
                            {"-", "@", "-", "@", "-", "@", "-", "-", "-", "@"},
                            {"-", "@", "@", "@", "-", "@", "-", "-", "-", "-"},
                            {"-", "@", "-", "@", "-", "@", "-", "@", "@", "@"},
                            {"-", "@", "@", "@", "@", "@", "-", "@", "@", "@"}};

At first, you might think that this is an easy problem and you can just iterate through the array. However, this method is extremely inefficient for larger arrays. Instead, it is much easier to solve this problem with a recursive depth-first search.

A depth-first search starts at an arbitrary spot in the array and recursively checks if all the spots around it are @ signs. If they are, a counter is incremented and the spot is temporarily changed to a dash to mark that it has already been searched. Here was the method that I wrote for this problem:

if (x >= 0 && x < blob.length && y >= 0 && y < blob[x].length && blob[x][y].equals("@"))
        {
            blob[x][y] = "-";
            count += 1;
            getCount(x - 1, y);
            getCount(x + 1, y);
            getCount(x, y- 1);
            getCount(x, y + 1);
        }
        return count;
