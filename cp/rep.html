This is the third problem in the CSES competitive programming problem set.

The problem goes like this: You are given a string that contains a DNA sequence, which is made up of the letters A, C, G, and T. Your task is to find the longest repetition in the sequence.

This problem is very straight forward and can be solved in O(n) fashion. Starting at the 2nd character of the string, compare all characters in the string from then on to the one before them 1 by 1. If it's the same as the previous, increment a counter. If it's different, reset this counter to 0. If the counter is ever above the max, which is 1 by default, set the max to the counter. Here's the code for this problem:

for (int j = 2; j <= test.length(); j++)
{
    if(test.substring(j - 1, j).equals(test.substring(j - 2, j - 1)))
    {
        count += 1;
        if (count > max)
        {
            max = count;
        }
    }
    else
    {
         count = 1;
    }
}
