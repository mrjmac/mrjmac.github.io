The hardest part of this problem was understanding what it was asking. Once I was able to do that though, it was very easy. For any given length from i to j, we need to find the total number of petals. Then, for each flower from i to j we need to check if the number of petals on that flower is equal to the mathematical average number of flowers from i to j. Here's the code for this problem: 

int photos = 0;
for (int i = 0; i < n; i++)
{
    for (int j = i; j < n; j++)
    {
        int totalPetals = 0;
        for (int k = i; k <= j; k++)
        {
            totalPetals += petals[k];
        }

        bool there = false;
        for (int k = i; k <= j; k++)
        {
            if(petals[k] * (j - i + 1) == totalPetals)
            {
                there = true;
            }
        }

        if (there)
        {
            photos += 1;
        }
    }
}


cout << photos << "\n";
