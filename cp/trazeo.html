Trailing Zeros

GitHub link

This is the tenth problem in the CSES problem set.

The problem goes like this: given a number n, calculate the number of trailing zeros in factorial n!.

This problem is 100% pure math. If you think about it, what we really need to find is the number of times that 5 is a factor of n. This is because the only way to arrive at another 0 at the end of your product is if the number is multiplied by 10. Five is in every pair that multiplies to any multiple of 10. However, just retuning n / 5 does not give the right answer. This is because some numbers like 25 are made up of two fives, adding two more uncounted fives to the mix. In order to solve this problem, I had to take into account all numbers that are exponents of 5. Here's the code for this problem:

for (int i = 5; n / i >= 1; i *= 5)
{
    ans += n / i;
}
