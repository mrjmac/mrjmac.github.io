The problem is rather long like all usaco problems, so here's a link to it so you can read it.

This problem is just a 2D version of Fence Painting. The solving principle of finding the intersection of the billboards remains the same here. However, this time we find two different lengths and multiply them in order to find the total area blocked. Originally my code for this was rather unreadable, however after getting all the test cases correct I went back and changed my code to use a struct as the usaco.guide solution used one to make it more readable. Here's the code for this problem:

struct board {
    int x1, x2, y1, y2;
    int area()
    {
        return (x2 - x1) * (y2 - y1);
    }
};

int overlap(board a, board b)
{
    int xO = max(min(a.x2, b.x2) - max(a.x1, b.x1), 0);
    int yO = max(min(a.y2, b.y2) - max(a.y1, b.y1), 0);
    return xO * yO;

}
