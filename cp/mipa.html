Milk Pails

This is an old usaco bronze problem that I found through usaco.guide.

This problem is rather long like all usaco problems, so here's a link so you can read it.

This problem was extremely easy. In ideal conditions, bucket x or y would perfectly fit into bucket m in m / x or y trips. However, it's far more likely that m is not perfectly divisible by x or y and instead is best filled by a combination of the two. In order to solve this, I just brute forced every possible combination of buckets until one filled m the most. 

for (int i = 0; i <= m / x; i++)
{
    for (int j = 0; j <= m / y; j++)
    {
        if ((x * i) + (y * j) <= m)
        {
            ans = max(ans, (x * i) + (y * j));
        }
    }
}

cout << ans << "\n";
