This problem is very straightforward. The original M x N image has to experience a dilation by a factor of K. All we have to do is use a nested loop to print out each character. For ever K iterations, we have to change the line or character that we print/reference. Here's the code for this problem:

int m, n, k;
string line;
string curr;
cin >> m >> n >> k;

for (int i = 0; i < m * k; i++)
{
    if (i % k == 0)
    {
        getline(cin >> ws, line);
    }
    for (int j = 0; j < n * k; j++)
    {
        if (j % k == 0)
        {
            curr = line.substr(j / k, 1);
        }
    cout << curr;
    }
    cout << "\n";
}
