Bovine Shuffle

GitHub link

This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem just involves simulating the shuffling of the cows backwards. In order to do this, we have to continually swap cows. However, whenever Farmer John originally swapped his cows, he did it in O(1) and each cow moved at the same time. However, we can only simulate this in O(n). This is a big problem as it can lead to numbers getting swapped twice. In order to fix this, I used a temporary array to store all the swaps and performed them all at the end of each shuffle. Here's the code for this problem:

for (int i = 0; i < 3; i++)
{
    int temp[n];
    for (int j = 0; j < n; j++)
    {
        temp[j] = tags[a[j] - 1];
    }

    for (int j = 0; j < n; j++)
    {
        tags[j] = temp[j];
    }
}
