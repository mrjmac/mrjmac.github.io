This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link so you can read it.

This problem is extremely trivial. The best case for the shell will always be equal to the shell that Bessie guessed the most. In order to figure out which shell this is, we just have to simulate the swapping of the shells and increment the shell that Bessie guesses. Then we just have to find the max value out of all 3 shells. Here's the code for this problem

vector<int> shells{0, 1, 2};
vector<int> correct(3);

for (int i = 0; i < n; i++)
{
    int a, b, g;
    cin >> a >> b >> g;

    a--, b--, g--;

    swap(shells[a], shells[b]);
    correct[shells[g]]++;
}

cout << max(max(correct[0], correct[1]), correct[2]) << "\n";
