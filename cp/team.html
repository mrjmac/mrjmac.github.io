This is an introductory codeforces problem that I found through usaco.guide.

The problem goes like this: The first line of input contains an integer n, the number of problems in a competitive programming contest. 3 friends are working together in this contest and have agreed to only solve a problem if the majority of them know how to solve it. The next n lines of input contain 3 integers, each of which reflects each of the friends view on the problem. Output the number of problems that the friends attempt.

This problem is also pretty easy. For each problem, we just check if the sum of all integers on the line is greater than 1. If it is, we increment a counter. Here's the code for this problem:

for (int i = 0; i < n; i ++)
        {
            st = new StringTokenizer(r.readLine());
            a = Integer.parseInt(st.nextToken());
            b = Integer.parseInt(st.nextToken());
            c = Integer.parseInt(st.nextToken());

            if (a + b + c > 1)
            {
                result += 1;
            }

        }
        pw.println(result);
