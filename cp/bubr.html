This is an old usaco bronze problem that I found through usaco.guide.

The problem is rather long like all usaco problems, so here's a link to it so you can read it.

This problem is rather daunting at first glance and one might assume that you have to use DFS to find the best path. However, this would make the problem way more complicated than necessary. The shortest travel distance for the cows will always be equal to the difference in x and y combined minus 1 (because the corner is one both the x and y) between the barn and the lake. If the rock is along this path, it only takes two extra cows to get around it(An extra to create an offset from the rock and go around it and one more to go back to the barn at the end. The solution that I opted for recorded the coordinate of all the objects and checking if the rock is in the way. Here's the code for this problem:

for (int y = 0; y < 10; y++)
{
    StringTokenizer st = new StringTokenizer(r.readLine());
    String a = st.nextToken();
    for (int x = 0; x < 10; x++)
    {

        if (a.charAt(x) == 'B')
        {
            BX = x;
            BY = y;
        }

        if (a.charAt(x) == 'R')
        {
            RY = y;
            RX = x;
        }

        if (a.charAt(x) == 'L')
        {
            LX = x;
            LY = y;
        }
    }
}

int ans = Math.abs(BX - LX) + Math.abs(BY - LY) - 1;

if (BY == LY && BY == RY && ((LX < RX && RX < BX) || (BX < RX && RX < LX)))
{
    ans += 2;
}
else if (BX == LX && BX == RX && ((LY < RY && RY < BY) || (BY < RY && RY < LY)))
{
    ans += 2;
}

pw.println(ans);
