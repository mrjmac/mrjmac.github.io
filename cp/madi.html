Maximum Distance

GitHub link

This is an old codeforces problem that I found through usaco.guide.

This problem goes like this: given anywhere from 3 to 5000 integer points on a coordinate plane, find the square of the maximum Euclidean distance between any of the two given points.

This problem was extremely easy. As n is at most 5000, we can easily use an O(n^2) loop and still stay within the time limit. In order to solve this problem, I just checked every possible answer and stored the greatest one. Here's the code for this problem:

for (int i = 0; i < n; i++)
{
    for (int j = i + 1; j < n; j++)
    {
        int disx = x[i] - x[j];
        int disy = y[i] - y[j];
        int pos = disx * disx + disy * disy;
        ans = max(pos, ans);
    }
}
cout << ans << "\n";
