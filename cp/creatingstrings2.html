Creating Strings (Revisited)

Original writeup

This is the fourteenth problem in the CSES problem set.

The problem goes like this: Given a string, your task is to generate all different strings that can be created using its characters. Print the number of strings that can be created and then print all strings that can be printed in alphabetical order.

Originally, I solved this problem using backtracking. However, since I now program in c++, I revisited this problem and did it using next_permutation. It feels like cheating. Here's the code for this problem:

int main()
{
    cin.tie(0)->sync_with_stdio(0);

    string s;
    cin >> s;

    vector<string> ans;
    sort(s.begin(), s.end());
    do
    {
        ans.push_back(s);
    }
    while (next_permutation(s.begin(), s.end()));

    cout << ans.size() << "\n";

    for (int i = 0; i < (int) ans.size(); i++)
    {
        cout << ans[i] << "\n";
    }
}
